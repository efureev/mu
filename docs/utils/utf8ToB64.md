# utf8ToB64

> <small>UTILS</small>

## Description

Encode string from Unicode to base-64.

## Use

```js
import utf8ToB64 from '@feugene/mu/src/utils/utf8ToB64.js'
```

## Examples

```js
utf8ToB64('1') // 'MQ=='
utf8ToB64('Hello world') // 'SGVsbG8gd29ybGQ='
utf8ToB64('Привет') // 'w5DCn8ORwoDDkMK4w5DCssOQwrXDkcKC'
```
