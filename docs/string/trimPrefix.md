# trimPrefix

> <small>STRING</small>

## Description

Remove a prefix from a String

## Use

```js
import trimPrefix from '@feugene/mu/src/string/trimPrefix.js'
```

## Examples

```js
trimPrefix('/es/test/es/page', '/es') // '/test/es/page'
trimPrefix('https://example.com', 'https://') // 'example.com'
```
