# µ

> A modern JavaScript collection of utilities designed for modularity & performance without dependencies.

## Install

`yarn add @feugene/mu`

## Links

- [Documentation](https://efureev.github.io/mu)

## Features

- TypeScript
- Bundles for
  - ES
  - CommonJS
  - UMD
- Simple and lightweight

## Contents

- Common helpers
- Array helpers
- Date helpers
- Format helpers
- Is-like helpers
- Number helpers
- Object helpers
- String helpers
- Structures
- Transform To
- Other Utils
  - Base64
- Etc
